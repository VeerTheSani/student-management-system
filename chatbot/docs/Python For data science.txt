7. Explain the use of head(), info(), and describe() functions.

These functions are used for data inspection in Pandas.

head()

Shows the first few rows of the dataset.

Example:

df.head()

Useful for quickly viewing data.

info()

Displays dataset structure.

Shows:

Column names
Data types
Missing values

Example:

df.info()
describe()

Provides statistical summary of numerical columns.

Shows:

Mean
Median
Standard deviation
Minimum and maximum values

Example:

df.describe()
8. What are missing values? Explain different methods to handle them.

Missing values occur when some data entries are empty or unavailable.

Example:

Name   Age
John   25
Sara   NaN

Methods to handle missing values:

1. Remove rows with missing values
df.dropna()
2. Replace with mean
df.fillna(df.mean())
3. Replace with median

Useful when data contains outliers.

4. Replace with constant value
df.fillna(0)
